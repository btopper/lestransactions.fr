L'API LesTransactions.fr

LesTransactions.fr dispose d'une API à laquelle vous pouvez accéder en faisant une requête POST sur cette page.
Ce service vous est proposé gratuitement, merci d'en faire un usage raisonnable.

Paramètres autorisés

- date avec le format yyyy-mm-dd, exemple : wget https://lestransactions.fr/api --post-data date=2017-06-30
Permet d'obtenir toutes les transactions pour une date donnée.

- isin avec le format FRXXXXXXX, exemple : wget https://lestransactions.fr/api --post-data isin=FRXXXXXXXX
Permet d'obtenir toutes les transactions pour un ISIN donné.

Réponse

La réponse est fournie au format json avec l'architecture suivante :
(1) une architecture qui numérote chaque résultat de 0 à N
(2) pour chaque résultat "i", les clés suivantes :
- "isin" : ISIN
- "company" : Nom de la société
- "manager" : Nom du dirigeant
- "date_transac" : Date de la transaction
- "nature" : Nature de la transaction (acquisition, cession, etc.)
- "instrument" : Instrument financier concerné (action, option, etc.)
- "price" : Prix unitaire
- "qty" : Volume d'actions concerné
- "total" : Montant total de la transaction
- "capital_share" : Pourcentage de la transaction par rapport à la capitalisation boursière
- "currency" : Monnaie
- "ref" : Lien vers le pdf de l'AMF
